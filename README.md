# movies-explorer-frontend

Репозиторий фронтенд части для приложения проекта "Movies-Explorer".

## Запуск проекта

Для запуска проекта необходимо выполнить следующие действия:

1. Склонировать проект
2. Установить зависимости:

```
npm install
```

3. Запустить проект:

```
npm start
```
